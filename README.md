# SNR-Clean-Noisy-Speech-Dataset
The dataset is organized into a main folder containing clean speech samples and noisy speech samples divided by SNR level. The noisy speech folders represent different noise conditions from -20 dB to +20 dB, making it easy to compare model performance across different noise intensities.
SNR Clean Noisy Speech Dataset

This repository contains a clean and noisy speech dataset created for speech enhancement, noise reduction, speaker recognition, and robustness testing under different Signal-to-Noise Ratio conditions.

The dataset includes clean speech recordings from 8 speakers and noisy speech samples generated at multiple SNR levels ranging from -20 dB to +20 dB.

Dataset Overview

The clean speech dataset was created using custom-recorded audio samples from male and female speakers. The noisy speech dataset was generated by mixing the clean speech samples with environmental noise taken from DEMAND and UrbanSound8K-style noise sources.

The purpose of this dataset is to provide controlled clean and noisy speech samples for testing machine learning and deep learning models under different noise conditions.

Key Features
Clean speech samples from 8 speakers
Male and female speaker recordings
5 clean samples per speaker
Noisy speech generated at multiple SNR levels
SNR range from -20 dB to +20 dB
Environmental noise collected from DEMAND and UrbanSound8K-style noise sources
Suitable for speech enhancement, denoising, speaker recognition, and audio robustness testing
SNR Levels

The noisy speech samples are generated at the following SNR levels:

-20 dB, -15 dB, -10 dB, -5 dB, 0 dB, 5 dB, 10 dB, 15 dB, 20 dB

Negative SNR values represent highly noisy conditions, while positive SNR values represent cleaner speech conditions.
Use Cases

This dataset can be used for:

Evaluation of models under different SNR levels
